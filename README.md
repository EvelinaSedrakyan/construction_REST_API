# Construction REST API

## Стек технологий
- Python 3.14
- FastAPI
- SQLAlchemy
- PostgreSQL
- Alembic
- Uvicorn

## Документация
- OpenAPI 3.1
- Swagger UI (`/docs`)

## Описание
REST API для управления строительной компанией.
Приложение поддерживает работу с компаниями, проектами и работниками,
включая CRUD-операции, фильтрацию, сортировку, агрегации, поиск по JSON
и работу с индексами.

## Заполнение базы данных
Заполнение базы данных выполнено с помощью Python-скрипта (`scripts/fill_data.py`),
который взаимодействует с приложением **исключительно через REST API**


В результате работы скрипта было создано:
- **50 компаний**
- **200 проектов**
- **1092 работника**
