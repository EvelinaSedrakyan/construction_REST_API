Construction REST API

Стек технологий: Python 3.14, FastAPI, SQLAlchemy, PostgreSQL, Uvicorn
Документация: OpenAPI 3.1 / Swagger UI

Описание:
REST API для управления строительной компанией. Позволяет работать с компаниями, проектами и работниками. Поддерживает CRUD операции, фильтрацию, поиск по JSON и отчеты.
